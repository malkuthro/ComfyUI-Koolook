The sidebar has six main areas.

Snapshot, Tools, Search, Nodes, Workflows, and saved lists work together to keep each project kit easy to save, find, and reload.

Most important idea

Snapshot = your project kit

A Snapshot saves Kforge Labs state: favorite nodes, saved workflows, modules, tags, archive, and the current sidebar organization. Use one named snapshot per project, shot, client, or experiment.

Project rhythm

A simple workflow

  1. Load or create a snapshot, for example Client_Shot010.
  2. Add favorite nodes while building the shot.
  3. Save reusable node clusters as modules.
  4. Quick Save during the session.
  5. Save named snapshots for each project, shot, or task.
  6. Load the right snapshot when you return.
