Easy Image Batch — behavior matrix demo

A live model of the proposed easy_ImageBatch logic. Frames are labelled like the Easy Pattern (Koolook) test node: S# = a source_batch frame (VFX number), I# = an keyframes_insert frame (ordinal), M# = a connected image1–4 override. A white ring = "placed" (counts toward alpha_batch / selected_*). Spec: easy-image-batch-behavior.md.

Inputs

frames 4
length
total_frames
cut_start_frame
placeholder
width height

Output

image_batch

alpha_batch (black = 0.0 / placed, white = 1.0 / empty)

source frame (S#) insert (I#) manual override (M#) placed (alpha 0)
