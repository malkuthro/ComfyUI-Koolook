# Demo workflows

Release-ready ComfyUI workflow JSONs that ship with Kforge Labs.

The sidebar-seeded copies live in [`../../web/workflow_defaults.json`](../../web/workflow_defaults.json)
so fresh installs can find them under **Kforge Labs Workflows** in the
Kforge Labs sidebar. The files here are the inspectable source copies for
maintainers, docs, and GitHub browsing.
